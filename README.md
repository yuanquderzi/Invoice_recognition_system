## **一、项目简介**
使用FastAPI构建发票识别系统后端服务，支持并发。使用ERFNet模型训练发票轮廓检测，进行畸变矫正，OCR识别，模板匹配，支持倾斜发票识别。准确率99.9%。
可输出json格式结果和csv格式结果。支持各种类型发票。
